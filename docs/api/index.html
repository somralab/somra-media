URL: https://github.com/somralab/somra-media License: AGPL-3.0-or-later

Somra — all-in-one, self-hosted home media server.

This document is the design-first source of truth for the Somra HTTP API. Frontend TypeScript types are generated from this file (see scripts/gen-openapi-types.sh and make openapi-types). The Go backend implements the contract defined here; any divergence is a bug in the implementation, not in this specification.

Errors follow a ProblemDetails-style envelope: every error response uses the ErrorResponse schema, which is a superset compatible with RFC 7807 "Problem Details for HTTP APIs" (the messageKey field carries an i18n key resolvable by the client; message carries the already-localized human text). See .plan/i18n-localization.md for the binding i18n rules.

Liveness/readiness probe

Reports the operational state of the Somra server and its critical dependencies. Intended for container orchestrators, monitoring systems and the setup wizard. Does not require authentication.
